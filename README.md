Jeu de Mots Glissés - Projet d'Algorithmique
Ce projet est un jeu de plateau développé en C# réalisé durant mon cursus à l'ESILV.

Points clés :
Algorithmique : Gestion d'un plateau de lettres et d'un dictionnaire de mots français.

Logique : Implémentation des règles de jeu et gestion des scores des joueurs.

Outil : Développé sous Visual Studio.
